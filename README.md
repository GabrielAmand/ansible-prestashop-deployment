## Ansible Vault

Secrets are stored using Ansible Vault.

To run the playbooks, you will be prompted for the vault password.

For demonstration purposes, a simple password can be used.

In a production environment, a strong password and a secure storage method (e.g. password manager or CI secrets) should be used.
